# Getting Started

## Prerequisites

- VSCode with PlatformIO

- checkout repository:

```
git clone --recurse-submodules https://github.com/heartwerktech/disko-star.git
```

## Build

- select env (wemos_d1_mini32)
- Build, Upload
- Build Filesystem Image
- Upload Filesystem Image
